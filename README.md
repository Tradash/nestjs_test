## Description

Test assignment for Node.js developer <br>
info testAassignment.md

## Installation

```bash
$ git clone https://github.com/Tradash/nestjs_test.git
$ cd nestjs_test
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
